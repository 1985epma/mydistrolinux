# Changelog

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),

## [Unreleased]

### Adicionado
- CI/CD com GitHub Actions para releases automáticas
- Sistema de versionamento semântico
- Validação automática de código com ShellCheck
- GRUB customizado com tema visual e múltiplas opções de boot
- Suporte para repositórios de terceiros (Chrome, Edge, Spotify, Steam)
- ISOLINUX customizado para boot BIOS

### Modificado
- README expandido com documentação completa
- Configurações de boot otimizadas

- N/A

## [1.0.0] - 2025-12-05

### Adicionado
- Script inicial para criação de distribuições Linux personalizadas
- Interface gráfica com Zenity
- Suporte para Ubuntu 22.04 e 24.04
- Ambiente desktop XFCE4
- Sistema de build completo com debootstrap
- Geração de ISO bootável (BIOS e UEFI)
